# Food Xpiry - Smart Food Waste Tracker

## Project Overview

Next.js application for tracking food expiration dates with AI-powered shelf-life estimates and storage tips.

**URL**: my-expiry.vercel.app
**GitHub**: github.com/tbrown034/food-xpiry

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
- **AI**: OpenAI/Claude for shelf-life estimates

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Log groceries with purchase dates
- AI-powered shelf-life estimates
- Storage tips and recommendations
- Expiration alerts
